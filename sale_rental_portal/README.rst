Sale Rental Portal
==================

.. contents:: Table of Contents

Description
-----------
This module separates rental orders from other sales orders on the portal.

.. image:: sale_rental_portal/static/description/portal_menu.png

My Orders
~~~~~~~~~
The section ``My Orders`` shows confirmed orders, except rental orders.

.. image:: sale_rental_portal/static/description/sales_order_list.png

.. image:: sale_rental_portal/static/description/sales_order_form.png

My Rentals
~~~~~~~~~~
The section ``My Rentals`` shows confirmed rental orders.

.. image:: sale_rental_portal/static/description/rental_order_list.png

.. image:: sale_rental_portal/static/description/rental_order_form.png

Quotations
~~~~~~~~~~
The section ``Quotations`` is unchanged.
It show both rental and non-rental quotations.
