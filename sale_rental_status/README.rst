Sale Rental Status
==================

This module adds a field Rental Status on the rental order which is displayed on form,
 list, kanban views

This field shows the status of delivery progress and return progress of the rental
order:
- Rented
- Partially Delivered
- Delivered
- Partially Returned
- Returned
